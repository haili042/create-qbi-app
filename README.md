# create-qbi-app
Quickly Create a Quick-BI custom component project. 
## Quick start

3 ways for you to getting start:
### npm
```
npm init qbi-app my-app
cd my-app
npm i
npm run start
```

### npx
```
npx create-qbi-app my-app
cd my-app
npm i
npm run start
```

### yarn
```
yarn create qbi-app my-app
cd my-app
yarn
yarn start
```
