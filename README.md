# create-qbi-app
Create Quick-BI custom components with no build configuration.


## Quick Overview

3 ways for you to getting start:
### npm
```
npm qbi-app my-app
cd my-app
npm i
npm run start
```

### npx
```
npx create-qbi-app my-app
cd my-app
npm i
npm run start
```

### yarn
```
yarn create qbi-app my-app
cd my-app
yarn
yarn start
```
